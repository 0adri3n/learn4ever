How to use learn4ever ?

Just add a memo, and it will be send between 10 and 20 minutes.
By see it many times, your brain will remember it easier that if you want to learn it in a short time.
So don't do it if you need your memo for the next day !
You can close the window, she'll be reduced as a tray notification. Right click on the tray to re-open it or quit it.

How to delete/modify a memo ?

Go in learn4ever files.
resources > app > db
Then delete the .json file with the ID of the memo that you want to delete. Or edit the .json file, like the title or others...

How I can share a memo ?

Go in learn4ever files.
resources > app > db
Then send the .json file of the memo that you want to share to someone. The receiver have to move the file in his files at resources > app > db. That's all !
